# 🏋️‍♂️ SportShop — Интернет-магазин спортивной экипировки

**Курсовой проект / выпускная квалификационная работа**  
*ASP.NET Core MVC | Entity Framework Core | SQLite | REST API | Bootstrap 5*

---

## 📌 Описание проекта

SportShop — это полнофункциональный веб-сайт для продажи спортивных товаров.  
Проект разработан на платформе **ASP.NET Core 8 (совместим с .NET 10)** с использованием архитектурного паттерна MVC, собственного Web API и базы данных SQLite.

### Возможности пользователя:
- Просмотр каталога товаров с фильтрацией по категориям (в планах)
- Детальная страница каждого товара
- Добавление товаров в корзину с выбором количества
- Просмотр корзины, изменение количества, удаление товаров, полная очистка
- Автоматический подсчёт общего количества товаров в корзине (счётчик в шапке)
- API для получения списка товаров, добавления, изменения и удаления (REST)

### Административные возможности (опционально):
- Через API можно управлять товарами (POST, PUT, DELETE) — можно расширить админ-панелью.

---

## 🛠 Технологии

- **Backend**: ASP.NET Core MVC + Web API, C# 12
- **ORM**: Entity Framework Core
- **БД**: SQLite (легковесная, не требует установки)
- **Frontend**: Razor Pages, Bootstrap 5, пользовательский CSS
- **API-документация**: Swagger (OpenAPI)
- **Хранение корзины**: сессии ASP.NET Core
- **Среда разработки**: Visual Studio 2022 / VS Code / Rider

## 🔧 Возможные проблемы и их решение

| Проблема | Решение |
|----------|---------|
| База данных не создаётся | Удалите `sportshop.db` и перезапустите проект |
| Счётчик корзины не обновляется | Проверьте регистрацию `AddSession()` и `app.UseSession()` в `Program.cs` |
| Не добавляется товар в корзину | Убедитесь, что в `CartController` метод `AddToCart` имеет атрибут `[HttpPost]` и принимает `id` |
| Стили не применяются | Очистите кэш браузера (Ctrl+F5) |
